# Medical-Clinic-Management
This is first big project for my studies
It's a medical clinic managment software which consists of two projects. First is windows forms application - this is part which employees of medical clinic use to manage it. Basic funcionalities: Adding new patients, Adding new employees, Assigning medical activities to employees, Assigning doctor's offices, Work hours, Adding and managing visits. Second part of program is simple web app - this part is made for clinic patients. Basic funcionalities: Logging in, Getting new appointment.
